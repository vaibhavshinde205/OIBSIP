#  Tasks Included

## Task 1 – Iris Flower Classification
A machine learning classification project that predicts iris flower species based on flower measurements.

### Features
- Data preprocessing  
- Label encoding of species  
- Train–test split  
- Random Forest Classifier  
- Confusion Matrix visualization  
- Feature Importance analysis   
 
---

##  Task 3 – Car Price Prediction
A regression model that predicts the selling price of used cars based on vehicle details.

### Features
- Data cleaning and preprocessing  
- Feature Engineering (Car Age)  
- Encoding categorical features  
- Random Forest Regressor  
- Feature Importance visualization  
- Actual vs Predicted price plot  

---

##  Task 4 – Email Spam Detection
A Natural Language Processing project that classifies emails as Spam or Not Spam.

### Features
- Text preprocessing  
- TF-IDF Vectorization  
- N-gram feature extraction  
- Naive Bayes Classifier  
- Confusion Matrix  
- Identification of important spam words

# How to Run

1. Open the project folder. 
2. Launch Jupyter Notebook or open the notebook in Google Colab. 
3. Open the `.ipynb` file for the task. 
4. Upload the required dataset file when prompted (if not already included). 
5. Click **Runtime → Run All** (Colab) or **Run → Run All Cells** (Jupyter).
